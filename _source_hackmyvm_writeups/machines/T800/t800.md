# T800

## Executive Summary
| Machine | Author | Category | Platform |
| :--- | :--- | :--- | :--- |
| T800 | sml | Beginner | HackMyVM |

**Summary:** T800 is a beginner-level machine that demonstrates classic web enumeration techniques, steganography analysis, and privilege escalation through misconfigured SUID binaries. The attack chain involves discovering credentials embedded in JPEG metadata, leveraging SSH access, and exploiting the calife utility to gain user privileges before finally using conky's SUID permissions to extract root SSH keys for complete system compromise.

## Attack Chain Summary

1. **Reconnaissance**: Network scanning identified target at 192.168.100.55 with ports 80 (HTTP) and 800 (SSH) open
2. **Web Enumeration**: HTML source revealed username "ruut" and background image rms2.jpg, robots.txt disclosed /sexy/ directory
3. **Directory Fuzzing**: FFUF discovered /sexy/nudeslut.jpeg containing embedded password in JPEG metadata
4. **Credential Extraction**: File analysis revealed password "ch[REDACTED]" embedded in JPEG comment field
5. **Initial Access**: SSH authentication as user "ruut" using discovered credentials on non-standard port 800
6. **User Escalation**: Exploited calife SUID binary to escalate to "superruut" user via vim shell escape (!/bin/bash)
7. **Root Escalation**: Leveraged conky SUID binary to read root's SSH private key, then authenticated as root via SSH key
