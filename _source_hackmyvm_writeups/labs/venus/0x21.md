Now, the challenge was getting that image off the server and onto my own machine where I could actually look at it. I saved the decoded bytes into a temp file:

```bash
iris@venus:~$ cat eloise | base64 -d > /tmp/pict.jpg
```

Then, switching back to my local PowerShell, I reached out across the network. Using scp on port 5000, I pulled that mystery file into my local drive. It’s always a satisfying feeling when you see that 100% transfer completion.

```powershell
PS D:\> scp -P 5000 iris@venus.hackmyvm.eu:/tmp/pict.jpg .
```
