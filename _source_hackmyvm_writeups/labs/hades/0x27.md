### 2. Understanding PTX - The Permuted Index Tool

The mission hint "permute" directly references the **ptx** command, which stands for **permuted index**. According to [GTFOBins](https://gtfobins.github.io/gtfobins/ptx/), `ptx` is a text processing utility that can be exploited to read arbitrary files when run with elevated privileges.

### 4. Exploiting PTX to Read the Password

Following the GTFOBins documentation for `ptx`, I used it to read the password file. The `-w` flag sets the output width to 99 characters for better readability:

```bash
demeter@hades:~$ sudo -u echo /usr/bin/ptx -w 99 /usr/local/games/echo_pass.txt
                                                    [REDACTED]
```

#### Explanation of the Command:
* **`sudo -u echo`**: Execute the command as user `echo`
* **`/usr/bin/ptx`**: The permuted index tool (listed on GTFOBins)
* **`-w 99`**: Set output width to 99 characters to display content clearly
* **`/usr/local/games/echo_pass.txt`**: The target file containing echo's password

### 6. Explanation

* **PTX (Permuted Index)**: A GNU text utility listed on [GTFOBins](https://gtfobins.github.io/gtfobins/ptx/) as a potential privilege escalation binary. It creates a permuted index of file contents but can also be exploited to read arbitrary files.
* **GTFOBins**: A curated list of Unix binaries that can be used to bypass local security restrictions. When checking sudo privileges, consulting GTFOBins is essential for finding exploitation techniques.
* **Mission Hint**: The word "permute" was a direct reference to the `ptx` command, which performs text permutation operations.
* **Sudo Misconfiguration**: Allowing users to run text processing utilities like `ptx` with elevated privileges can lead to information disclosure, as these tools can read files accessible to the target user.
* **File Reading via PTX**: While designed for creating word indexes, `ptx` can effectively display file contents, making it dangerous when granted via sudo.
