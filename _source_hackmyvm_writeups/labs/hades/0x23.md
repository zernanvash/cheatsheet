### 3. Steganography Analysis: LSB in Red Channel

The mission hint suggests cynthia "sees things that others don't," which is a common reference to **steganography** — hidden data embedded within image files.

I used ImageMagick's `convert` command to isolate the **least significant bit (LSB)** of the **red channel**:

```bash
┌──(ouba㉿CLIENT-DESKTOP)-[/mnt/d/hackmyvm-writeups]
└─$ convert fun.png -channel R -separate -evaluate and 1 -negate -monochrome red0_isolated.png
```

#### Explanation of the Command:
* **`-channel R -separate`**: Extracts only the red channel from the image
* **`-evaluate and 1`**: Performs a bitwise AND operation with 1, isolating only the LSB (the rightmost bit)
* **`-negate -monochrome`**: Inverts and converts to black and white for visibility

### 6. Explanation

* **LSB Steganography**: This technique hides data by replacing the least significant bits of pixel values in an image. These changes are imperceptible to the human eye but can encode messages.
* **Red Channel LSB**: The password was hidden specifically in the LSB of the red color channel, requiring specialized extraction to reveal.
* **Mission Hint**: "Sees things that others don't" was a direct reference to the hidden steganographic data that requires special tools to uncover.
* **ImageMagick**: A powerful image manipulation toolkit that can extract and analyze individual color channels and bit planes.
