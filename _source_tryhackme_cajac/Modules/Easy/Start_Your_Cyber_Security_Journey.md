# Start Your Cyber Security Journey Module

```text
Type: Module
Difficulty: Easy
Description:
Explore offensive and defensive cyber security via interactive exercises and acquire 
the essential search skills to find information.

This module introduces the reader to offensive cyber security through an interactive 
exercise where users can practice hacking an insecure web application in a legal and 
safe environment. Then, it presents defensive security via an interactive exercise 
to protect against an attack. Finally, it equips the reader with the fundamental 
search skills to find what they need as they start their journey in cyber security.
```
