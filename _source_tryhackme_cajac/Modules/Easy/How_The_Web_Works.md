# How The Web Works Module

```text
Type: Module
Difficulty: Easy
Description:
To become a better hacker it's vital to understand the underlying functions 
of the world wide web and what makes it work.

In this module, we'll take you through the building blocks of the world wide 
web and explain in detail exactly what happens when you enter a website 
address into your browser. From DNS to HTTP protocol this module will give 
you all the information you need to understand the world wide web.
```
