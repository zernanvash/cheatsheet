# Metasploit Module

Description:
Metasploit is the most widely used exploitation framework. Learn how to use it and 
unlock its full potential.

The Metasploit framework is a set of open-source tools used for network enumeration, 
identifying vulnerabilities, developing payloads and executing exploit code against 
remote target machines. Get hands-on with the various tool and features Metasploit 
provides, from exploit development to post-exploitation techniques, this module covers it all.
