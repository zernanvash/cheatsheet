# Network Traffic Analysis Module

```text
Type: Module
Difficulty: Easy
Description:
Learn the basics of network traffic analysis and get hands-on with Wireshark to detect various 
types of attacks.

This module explores the basics of Network Traffic Analysis. Discover what Network Traffic 
Analysis is, why you need it, how to collect the network traffic data to analyse, and how to 
analyse the network traffic. Learn the basics of Wireshark and apply them to discover packet 
operations and detect typical attacks like ARP poisoning, nmap scans, ICMP tunnelling and more.
```
