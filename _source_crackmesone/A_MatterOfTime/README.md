# Description

Encrypted flag with AES-CBC-128, the key being the discoverable username and the IV being an amalgamated value from UNIX decimal timestamps. The player has to use an educated brute force to find the flag. It doesn't take so long; you just need to know the right format. Also, the binary is obfuscated with my own bin2bin. Not so difficult to deobfuscate.

# Supported OS

I primarily worked on Windows 10. So it prolly works on any Windows10+ version.
